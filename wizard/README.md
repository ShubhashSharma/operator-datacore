# wizard/

Reserved for a hosted web wizard (Vercel + Next.js) that walks brand-new sellers through Supabase project creation and credential collection — for use cases where the seller doesn't have Claude Code installed.

**v1 ships nothing here.** The Claude Code `/operator-setup` slash command is a better experience for SSL 2026 attendees (everyone has Claude Code installed). When we open this up to sellers without an IDE, that's the home for a Vercel-hosted setup form.

Tracked as a homework item in [HOMEWORK.md](../HOMEWORK.md).
